Git Команды

1. Что такое git fetch ?

Команда git fetch используется для загрузки коммитов и файлов из удаленного репозитория в ваш репозиторий

2. Разница между git fetch и git pull

git fetch сливает все данные с проекта а git pull загружает ветку из удаленного репозитория

3. Что такое git merge ?

Команда git merge используется для объединения двух веток

4. Что такое git rebese?

Команда git rebaseиспользуется для объединения двух веток.

5. Разница между git merge и git rebese

git merge создает коммит слияния, git rebase соеденяет две ветки

Git Workflow

Git Workflow - это модель управления ветками

В Git Workflow создается 5 веток

  • Main - ветка на которой ваш проект
  • Develop - основная ветка разработки
  • Feature - ветка для разработки новых функций, которые далее сливаются с develop
  • Release - ветка, которая создается, при завершении проектаи и с этого момента новые функции добавить больше нельзя
